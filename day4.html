DAY 04

片假名 ハ行 - ン + 特殊音

完成片假名學習，掌握長音、促音、拗音等特殊發音規則

一、課程介紹

今天我們將完成片假名的學習！學習 ハ行、マ行、ヤ行、ラ行、ワ行、ン 以及濁音、半濁音。

此外，還會學習片假名特有的 長音（ー）、促音（ッ）、拗音 等特殊發音規則，這些在外來語中非常常見。

今日學習重點

1. 片假名的濁音、半濁音寫法與平假名相同

2. 長音符號「ー」用於延長前一個音

3. 促音「ッ」（小寫的ツ）表示短暫停頓

4. 拗音由大假名 + 小假名（ャ、ュ、ョ）組成

八、長音（ー）

長音符號「ー」用於延長前一個假名的母音，發音時間是普通音的兩倍。這是片假名特有的符號。

長音規則與例子
コーヒー → ko-o-hi-i (咖啡 coffee)
カード → ka-a-do (卡片 card)
ケーキ → ke-e-ki (蛋糕 cake)
スーパー → su-u-pa-a (超市 supermarket)

九、促音（ッ）

促音「ッ」是小寫的ツ，表示短暫的停頓，類似英文的雙子音。發音時在該處停頓一拍。

促音規則與例子
ベッド → be-(停頓)-do (床 bed)
サッカー → sa-(停頓)-ka-a (足球 soccer)
ロケット → ro-ke-(停頓)-to (火箭 rocket)
カップ → ka-(停頓)-pu (杯子 cup)
促音重點

促音「ッ」必須寫得比正常的ツ小，位置偏右下

促音後面通常接 k, s, t, p 行的假名

十、拗音

拗音由一個大假名 + 小假名（ャ、ュ、ョ）組成，兩個假名合併發一個音。

常見拗音組合
キャ キュ キョ → kya, kyu, kyo
シャ シュ ショ → sha, shu, sho
チャ チュ チョ → cha, chu, cho
ニャ ニュ ニョ → nya, nyu, nyo
シャツ shatsu
襯衫 (shirt)
ジュース juusu
果汁 (juice)
チョコレート chokoreeto
巧克力 (chocolate)
